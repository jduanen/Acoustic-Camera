# Acoustic Camera — Phase 3 runtime dependencies (live script + benchmark)

# On the Raspberry Pi deployment (see RASPBERRY_PI.md):
#   - create the venv with --system-site-packages
#   - cv2 comes from apt (python3-opencv — the pip wheel is often headless on ARM)
#   - picamera2 (for --csi) comes from apt (python3-picamera2), not pip
#   - OpenBLAS must be installed: sudo apt install libopenblas0-pthread
# On a desktop, `pip install -r requirements.txt` inside a plain venv is enough.
